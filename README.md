# LLMEmbed Fraud Experiments Code-Only Package

This package contains code and reproducibility instructions only. It does not include generated tensors, checkpoints, logs, prediction files, rewrite JSON outputs, or pretrained model weights.

## What You Need To Provide

Prepare two Hugging Face `datasets.load_from_disk` directories:

```text
dataset/fraud_binary
dataset/fraud_multi
```

Both datasets should contain `train` and `test` splits with `text` and `label` columns. `fraud_binary` is the binary fraud detection task. `fraud_multi` contains fraud samples with fraud type labels and is aligned with positive fraud samples in `fraud_binary/test`.

Download or place encoder models here:

```text
hf_model/bert-large-uncased
hf_model/roberta-large
hf_model/llama2-7b-chat-hf
```

You can also use custom paths:

```bash
export DATASET_ROOT=/path/to/datasets
export BERT_MODEL_PATH=/path/to/bert-large-uncased
export ROBERTA_MODEL_PATH=/path/to/roberta-large
export LLAMA2_MODEL_PATH=/path/to/llama2-7b-chat-hf
```

For adversarial rewrite generation, use either local Ollama or an OpenAI-compatible API. The rewrite model outputs are generated by `adversarial_rewrite/rewrite_with_llm.py`.

## Environment

```bash
conda create -n llmembed python=3.9
conda activate llmembed
pip install -r requirements.txt
```

The scripts whitelist CUDA ids `0`, `1`, and `2`. If your machine uses different ids, update `ALLOWED_CUDA_IDS` in the scripts or run on matching ids.

## 1. Generate Baseline Embeddings

Run all three encoders for both fraud tasks:

```bash
python bert_embedding/rep_extract.py 0 fraud_binary
python roberta_embedding/rep_extract.py 0 fraud_binary
python llama2_embedding/rep_extract.py 0 fraud_binary

python bert_embedding/rep_extract.py 0 fraud_multi
python roberta_embedding/rep_extract.py 0 fraud_multi
python llama2_embedding/rep_extract.py 0 fraud_multi
```

Expected outputs:

```text
bert_embedding/fraud_binary/dataset_tensor/
roberta_embedding/fraud_binary/dataset_tensor/
llama2_embedding/fraud_binary/dataset_tensor/
bert_embedding/fraud_multi/dataset_tensor/
roberta_embedding/fraud_multi/dataset_tensor/
llama2_embedding/fraud_multi/dataset_tensor/
```

## 2. Train Baseline Models

```bash
python main.py 0 fraud_binary 10 0.1 1024 0.0001 --seed 42
python main.py 0 fraud_multi 10 0.1 1024 0.0001 --seed 42
```

Checkpoints are saved under `checkpoints/`.

## 3. Export Attack Source

```bash
python adversarial_rewrite/export_attack_source.py
```

Expected outputs:

```text
adversarial_data/source/fraud_test_attack_source.json
adversarial_data/source/fraud_test_attack_source.summary.json
```

## 4. Generate Rewrite JSON Files

Example with local Ollama native API:

```bash
python adversarial_rewrite/rewrite_with_llm.py \
  --provider ollama-native \
  --model qwen3.5:9b \
  --json_mode \
  --temperature 0 \
  --timeout 300 \
  --retries 2 \
  --save_every 1 \
  --resume \
  --output adversarial_data/rewrites/qwen3.5-9b_fraud_test_rewrites.json
```

Repeat for the rewrite models you want to evaluate. The experiment commands used in this project are documented in `EXPERIMENT_COMMANDS.md`.

Validate generated rewrite files:

```bash
python adversarial_rewrite/validate_rewrites.py \
  adversarial_data/rewrites/qwen3.5-9b_fraud_test_rewrites.json \
  --summary_json adversarial_data/validated/rewrite_validation_summary.json \
  --summary_md adversarial_data/validated/rewrite_validation_summary.md
```

## 5. Build Adversarial Tensors

For each rewrite JSON and each encoder:

```bash
python adversarial_rewrite/build_adv_dataset.py \
  --rewrite_json adversarial_data/rewrites/qwen3.5-9b_fraud_test_rewrites.json \
  --rewrite_model_slug qwen3.5-9b \
  --encoder bert \
  --cuda_no 0

python adversarial_rewrite/build_adv_dataset.py \
  --rewrite_json adversarial_data/rewrites/qwen3.5-9b_fraud_test_rewrites.json \
  --rewrite_model_slug qwen3.5-9b \
  --encoder roberta \
  --cuda_no 0

python adversarial_rewrite/build_adv_dataset.py \
  --rewrite_json adversarial_data/rewrites/qwen3.5-9b_fraud_test_rewrites.json \
  --rewrite_model_slug qwen3.5-9b \
  --encoder llama2 \
  --cuda_no 0
```

Verify tensor consistency:

```bash
python adversarial_rewrite/verify_adv_tensors.py \
  --models qwen3.5-9b \
  --summary_json adversarial_data/validated/adv_tensor_summary.json
```

## 6. Evaluate Robustness

Replace checkpoint names with your generated files:

```bash
python adversarial_rewrite/evaluate_adv.py \
  --binary_checkpoint checkpoints/<fraud_binary_checkpoint>.pt \
  --multi_checkpoint checkpoints/<fraud_multi_checkpoint>.pt \
  --models qwen3.5-9b \
  --cuda_no 0 \
  --save_predictions
```

Outputs:

```text
adversarial_data/evaluation/adv_evaluation_summary.json
adversarial_data/evaluation/adv_evaluation_summary.md
adversarial_data/evaluation/predictions/
```
